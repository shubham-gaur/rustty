Features

Everything you need,
nothing you don't

Keedown is built around a single idea: your notes should be fast to write, easy to navigate, and powerful enough to replace a terminal.

✦

WYSIWYG Markdown

Write markdown and see it rendered live. No split panes — the source and the output are one. Swap to raw mode anytime.

⌘

Slash Commands

Type / to insert headings, lists, callouts, tables, kanban boards, carousels, images, and file attachments — all from the keyboard.

$_

Inline Shell Execution

Run any command inline and insert the output as a fenced block. Works with the local shell, SSH, GCloud IAP, and AWS SSM.

⎋

Integrated Terminal

Full xterm.js terminal with multiple sessions per tab, custom fonts, colours, and scrollback. SSH, tmux, and PTY — all first-class.

☁

SSH Binding

Bind a tab to an SSH profile (key, password, certificate, GCloud IAP, or AWS SSM) and inline commands execute remotely, not locally.

◑

Themes & Customisation

Light, Dark, Catppuccin, Dracula, GitHub, and GitHub Dark presets. Full accent colour, background, and font control.

⇥

Tab Groups

Open as many files as you need. Tabs remember their SSH binding, terminal sessions, search highlights, and scroll position.

🔒

Cryptographic Licensing

Time-bound Ed25519-signed licences. Verified at startup with no network call — fully offline, tamper-proof.

Editor

Rich blocks, right in your document

Insert structured content without leaving the editor. Every block is stored as plain Markdown — no proprietary formats.

Slash menu

Hit / from anywhere and search across all block types. Keyboard-navigable, grouped, and instant.

  • Headings, bullet, numbered, and task lists
  • Blockquotes and horizontal dividers
  • Fenced code blocks with syntax highlighting
  • Tables with a visual grid picker
  • Callout blocks (Info, Warning, Critical, Tip)
  • Kanban board, image carousel, file attachments
Callout

Info / Warning / Critical / Tip

Colour-coded callout blocks that render as styled banners and round-trip through standard Markdown fences.

Kanban

Drag-and-drop board

Full Kanban with multiple columns, drag-and-drop card reordering, and per-card rich text. Stored as a JSON fence.

Carousel

Slide gallery

Paginated image + caption slides. Copy, cut, paste, and reorder slides. Images stored as internal references.

Table

GFM tables with toolbar

Full table editing with a floating toolbar: add/delete/reorder rows and columns, and convert to plain text.

Image

Paste or browse

Drag, paste, or browse images. They are stored internally (no external hosting) and render with a resize handle.

Attachment

Inline file previews

Embed images, PDFs, and videos inline. Zoom to full-screen with a single click.

Inline Commands

Run commands, insert output

Write a command anywhere in your document and execute it with a keyboard shortcut. The result lands as a fenced shell block, right below — no copy-paste required.

  • Works with any shell command or script
  • Routes through the active SSH / terminal binding
  • 30-second timeout; output is trimmed and clean
  • Lock a terminal session to guarantee command routing

Terminal & SSH

Connect anywhere, run anything

Keedown has a full PTY terminal built in. Bind a tab to any remote host — commands you run in the editor execute there.

Native PTY, zero overhead

The terminal is an xterm.js view wired directly to a Rust PTY via Tauri IPC. No HTTP, no WebSocket — just raw bytes.

  • Multiple terminal sessions per tab
  • Custom font family, size, and scrollback
  • Lock a session to route all inline commands through it
  • Full ANSI / VT100 / mouse support
SSH Key / Cert

Standard SSH

Identity file or certificate auth to any OpenSSH-compatible host.

Password

Password auth

Plain password login for hosts that don't accept key auth.

GCloud IAP

Google Cloud IAP

Tunnel through Identity-Aware Proxy — no public IP needed. Supports interactive PTY sessions.

AWS SSM

AWS Systems Manager

Session Manager shell access to EC2 without opening port 22.
